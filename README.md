#
> [!Caution]
> Программа создана для настройки свежеустановленной системы. При запуске на рабочей / настроенной / оптимизированной ОС ответственность за возможные проблемы ложится на вас.

> [!WARNING]
> Рекомендуется использовать Windows 11 Pro (25H2+). Подключение к интернету должно отсутствовать для беспроблемного удаления браузера Edge и WebView2, а также Защитника.

> [!TIP]
> Установили Windows 11 без интернета > Настроили и запустили WinClick > Подождали > Получили чистую систему.
#
<h3 align="center">
  Полный перечень разделов и их твиков / настроек:
</h3>

### 🗑️ Очистка
- Удалить файлы обновлений
`(Файлы скачанных обновнений для Windows и браузера Edge)`
- Удалить кэш Microsoft Store
- Удалить кэш Проводника
- Очистить хранилище WinSxS
`(Удалит устаревшие и заменненные версии компонентов)`
- Удалить лишние папки на диске С
`(Windows.old, PerfLogs и inetpub)`
- Удалить старые драйверы
- Удалить ShellBags
> Последние два пункта добавлены на случай, если запускаете WinClick не на новой системе.
#
### 📦 Предустановленные приложения
- Удалить все UWP-приложения
`(Все программы в Настройки > Приложения > Установленные приложения, кроме Edge, WebView2, OneDrive и Подключение к удаленному рабочему столу)`
- Удалить OneDrive
- Удалить Подключение к удаленному рабочему столу
- Удалить лишние папки приложений в Пуске
`(Удаляет папки "Специальные возможности" и "Стандартные - Windows")`
> [!IMPORTANT]
> При удалении всех UWP-приложений также удалится Microsoft Store.

> [!Tip]
> Восстановить магазин приложений можно командой в командной строке: `wsreset -i`
#
### 🌐 Браузер Edge и WebView2
- Удалить Microsoft Edge
- Удалить Edge WebView2
> [!IMPORTANT]
> Если утилита "зависла" на этом пункте, значит есть подключение к сети, что крайне не рекомендуется. Завершите все процессы Edge WebView2 через Диспетчер задач.

> [!Tip]
> Портативные браузеры [Cent](https://www.centbrowser.com/), [CatsXP](https://www.catsxp.com/), [Floorp](https://floorp.app/).
#
### 🛡️ Защитник Windows
- Удалить Защитник Windows
> [!IMPORTANT]
> Удаление происходит при помощи утилиты [DefenderKiller](https://t.me/martyfiles/1504).
> Требуется лишь следовать несложным инструкциям в окне программы.

> [!Tip]
> Ярлык приложения Безопасность Windows в меню Пуск удаляется отдельно в DefenderKiller > Пункт 3 > Пункт 2.
#
### 🧩 Компоненты Windows
- Удалить все предустановленные компоненты
`(Удаляет все компоненты в Настройки > Система > Дополнительные компоненты > Посмотреть компоненты)` 
> [!IMPORTANT]
> Компонент `Язык сценариев VBScript` может быть нужен для установки некоторых программ (Epic Games) и драйверов (AMD).

> [!Tip]
> Если столкнулись с проблемами – установите его в `Настройки > Система > Дополнительные компоненты > Посмотреть компоненты > Посмотреть доступные функции`.
#
### 🕒 Планировщик задач
- Отключить задачи телеметрии и проверок
`(Отключает десятки задач связанных с телеметрией, проверками, отчетами и диагностиками)`
> [!Tip]
> Включить нужные задачи можно через Планировщик задач.
#
### ⚙️ Оптимизация параметров
- Отключить Гибернацию
- Отключить Зарезервированное хранилище
- Отключить Точки восстановления
- Отложенный запуск автоматических служб
- Минимизировать системные отчеты
- Увеличить кэш иконок
- Увеличить порог разделения SVC
- Ускорить открытие папок
- Отключить VBS и HVCI
`(Бустит производительность процессоров AMD)`
- Отключить Game DVR
- Установить схему питания "Максимальная производительность"
- Отключить функцию "Возобновить"
`(освобождает 200 МБ ОЗУ)`
> [!Tip]
> Вместо точек восстановления рекомендую делать [полноценный бэкап системы](https://www.youtube.com/watch?v=rHg_kV5xJnk).
#
### 🔄 Центр обновления Windows
- Запретить установку драйверов из ЦО
- Запретить обновления удаления вредоносных программ
- Установить паузу обновлений до 07.07.2077
- Запретить автоматические обновления
> [!Tip]
> Возобновить апдейты можно в Центре обновления Windows.
#
### 🛠️ Полезные твики
- Отключить UAC
- Сделать учетную запись Административной `(Всё будет открываться от админа, пункт "Запуск от имени Администратора" будет удален)`
- Снять региональные ограничения
- Принудительно завершать программы при зависании
- Отключить Удаленный помощник
- Отключить залипание клавиш
- Скрыть реальный TTL
`(Полезно при раздаче интернета через смартфон)`
- Отключить лишние уведомления и рекомендации
`(В Настройки > Система > Уведомления, Настройки > Персонализация > Пуск и Проводнике)`
- Установить DNS Cloudflare на Wi-Fi адаптеры
`(При проблемах с интернетом поменяйте DNS-сервер в свойствах сетевого адаптера)`
#
### 📥 Драйверы
- Установить драйверы
> [!IMPORTANT]
> Установка чистых драйверов происходит только при наличии папки Drivers на Рабочем столе.
> Если ее нет, WinClick пропустит этот этап.

> [!Tip]
> Экспортировать чистые драйверы можно через [WinClean](https://t.me/martyfiles/1596), [Win10Tweaker](https://win10tweaker.ru) и [DriverStoreExplorer](https://github.com/lostindark/DriverStoreExplorer).
>
> Подробнее о чистых драйверах в [видео](https://www.youtube.com/watch?v=lWFLBNbNZTU).
#
### 🕹️ Другие компоненты
- Установить Visual C++ 2005-2022
- Установить DirectX 9-11
> Компоненты будут скрыты в Установленных приложениях
#
### ✨ Визуальные твики
- Удалить пункт Главная в Проводнике
- Удалить пункт Галерея в Проводнике
- Удалить пункт Сеть в Проводнике
- Установить темную тему системы
- Установить кастомные обои
- Установить синие папки
- Установить дополнительные эскизы медиафайлов [Icaros](https://github.com/Xanashi/Icaros)
- Установить секунды в трее
- Установить дату в трее
- Установить пункт "Завершить задачу" на Панели задач
- Удалить лишние значки на Панели задач
`(Поиск, Представление задач, Виджеты)`
- Скрыть раздел Рекомендуем в меню Пуск
- Установить значок Настройки в меню Пуск
- Удалить сжатие обоев Рабочего стола
- Скрыть раздел Рекомендуем в меню Пуск
- Отключить экран блокировки
- Отключить тень на значках Рабочего стола
- Открывать Проводник в "Этот компьютер"
- Показывать расширения файлов
